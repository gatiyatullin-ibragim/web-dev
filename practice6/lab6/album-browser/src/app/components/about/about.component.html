About Album Browser

📋 About the Application

Album Browser is a single-page application (SPA) built with Angular 17. It demonstrates modern front-end development techniques including routing, HTTP communication with a REST API, reactive programming with Observables, and a clean service-layer architecture.

⚙️ Technologies Used

Angular 17
TypeScript
Angular Router
HttpClient
RxJS / Observables
JSONPlaceholder API

🎯 Features

  • ✅ Multi-route SPA with Angular Router
  • ✅ REST API integration via HttpClient
  • ✅ Album listing, detail view, and photo gallery
  • ✅ CRUD operations: Read, Update, and Delete
  • ✅ Reactive data handling with Observables
  • ✅ Loading indicators and empty state handling
  • ✅ Responsive photo grid layout
